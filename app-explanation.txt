Vet App Overview
What It Does
This is a veterinary clinic management system built with Flask that manages pets, owners, and vet appointments. It integrates with external dog and cat breed APIs and aims to track pet health metrics.

Architecture & Components
1. Core Framework
Flask - REST API backend
JWT Authentication - token-based user authentication
Flask-CORS - cross-origin requests support
SQLAlchemy - ORM for MySQL database
MySQL - primary database
2. Key Layers
Layer	Files	Purpose
Routes	user_routes.py, pet_routes.py	HTTP endpoints for users & pets
Services	auth_service.py, user_data_service.py, animal_data_service.py	Business logic, authentication, external API integration
Models	user.py, pet.py, dog.py, cat.py, animal.py	Data structures & inheritance hierarchy
Database	repo.py, db_config.py	Database connection pool & repository queries
Utils	auth_utils.py	Helper functions (password generation, etc.)
3. Current Endpoints
POST /users/authenticate - User login (returns JWT tokens)
GET /users/current - Get current authenticated user
GET /users/all - List all users (requires JWT)
POST /users/add - Create new user (requires JWT)
Pet routes skeleton exists but not implemented
